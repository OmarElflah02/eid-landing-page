# Eid Landing Page 🌙

A simple **Eid Al-Fitr themed landing page** built using **HTML and CSS**.

## 📌 Project Idea
This project was created to celebrate **Eid Al-Fitr** and practice building a modern and responsive landing page layout.

## 🚀 Features
- Responsive Design
- Clean UI Layout
- Hero Section
- About Section
- Gallery Section
- Video Section
- Contact Form
- Smooth Scrolling
- Mobile Friendly

## 🛠️ Built With
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- Font Awesome
